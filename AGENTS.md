
# FV Finance Lab — Project Instructions

## Project goal
Build a personal financial analytics web app that showcases practical tools and projects in finance, risk, derivatives, and fixed income.

## Current scope
This is the first version of the website.
Focus only on:
- clean structure
- dark professional UI
- reusable components
- placeholder pages where needed

## Required routes
- /
- /projects
- /tools
- /tools/options
- /tools/risk
- /tools/bonds

## Tech stack
- Next.js
- TypeScript
- Tailwind CSS
- App Router
- No backend for now
- No external APIs for now

## Rules
- Keep the UI dark, minimal, and professional
- Do not add dependencies unless strictly necessary
- Prefer reusable components
- Keep financial logic separate from UI
- Put financial calculations inside `src/lib/finance/`
- Do not implement features outside the requested scope
- Do not rewrite unrelated files
- Explain which files were changed after each task

## Initial product direction
This website is not a generic portfolio.
It is a finance tools platform.

## Planned tools
- Option Pricing Tool
- Portfolio Risk Tool
- Bond Calculator
